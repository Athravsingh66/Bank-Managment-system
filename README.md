# ATM Simulator (Python)

A simple ATM Simulator built using Python and basic programming concepts such as variables, conditional statements (if-else), and user input handling.

# Features
- PIN verification for secure access
- Check account balance
- Deposit money
- Withdraw money
- Exit the system

# Technologies Used
- Python

# About the Project
This project was created while learning Python fundamentals. It simulates basic ATM operations through a menu-driven interface and demonstrates the use of decision-making statements, user interaction, and simple banking logic.
